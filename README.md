# Insurance Boss Chatbot

A premium, embeddable chat widget for `theinsuranceboss.com`.

## Quick Start
1. Add your Gemini API key in `main.js`.
2. Open `index.html` in a browser.

## Integration
Add the following to your website:

```html
<!-- Styles -->
<link rel="stylesheet" href="style.css">

<!-- Widget -->
<div id="insurance-boss-widget">
    <!-- ... contents of index.html ... -->
</div>

<!-- Scripts -->
<script src="main.js"></script>
```

## Customization
- **Avatar**: Replace `avatar.png` with your preferred image.
- **Colors**: Adjust CSS variables in `style.css`.
- **Knowledge**: Update `knowledge_base.json` to add more info.
